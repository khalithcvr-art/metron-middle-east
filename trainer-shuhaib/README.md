# Trainer Shuhaib — Elite Personal Training Website

A single-page marketing site for Trainer Shuhaib, a Dubai-based personal trainer.
Built with React 19, TypeScript, Vite 7, Tailwind CSS 4, Framer Motion, and Lucide icons.

## Sections

- **Hero** — parallax headline, animated stats, gym visual with play overlay
- **Keyword marquee** — two infinite-scrolling rows of training specialties (opposite directions)
- **Social proof** — featured-by logo bar
- **About** — coach bio, certifications, animated count-up counters (clients, sessions, years, success rate)
- **Features** — coaching system pillars
- **Programs** — four signature training programs
- **The Method** — animated four-step process timeline
- **Gallery** — six training-style photos with hover zoom and caption reveals
- **Benefits** — "more than a trainer" grid with floating stat card
- **Nutrition & Recovery** — meal planning, supplementation, sleep, and mindset pillars with floating macro card
- **Testimonials, Pricing (AED), FAQ, Final CTA, Footer** — plus a floating WhatsApp button

## SEO

`index.html` includes an extended keyword set (personal trainer Dubai, fat loss, muscle
building, online coaching, etc.), Open Graph and Twitter Card tags, a canonical URL,
and JSON-LD `HealthClub` structured data.

## Develop

```bash
npm ci
npm run dev      # local dev server
npm run build    # type-check + production build to dist/
npm run preview  # serve the production build
```
